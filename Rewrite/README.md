# 使用方法

圈X 中添加重写资源，资源路径如下：

* 导入所有重写规则

  ```
  https://raw.githubusercontent.com/gaohongkui/QXConfig/main/Rewrite/UnlockApp.conf
  ```
* 导入 GodK 自用规则

  ```
  https://raw.githubusercontent.com/gaohongkui/QXConfig/main/Rewrite/GodKUsed.conf
  ```
